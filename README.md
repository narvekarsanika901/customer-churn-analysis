# Customer Churn Analysis

This project analyzes customer churn using Python.

Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn

Project Description:
The goal of this project is to analyze customer churn data and identify patterns that help businesses understand why customers leave.

Steps Performed:
1. Data loading using Pandas
2. Data cleaning and preprocessing
3. Exploratory Data Analysis (EDA)
4. Data visualization using Matplotlib and Seaborn

Dataset:
customer_churn.csv

Output:
Graphs showing churn patterns based on customer behavior.
